Mini Loja

31/01/2026

Um site exibindo um layout de loja virtual, o qual é simplesmente estético. Ele foi criado como forma de praticar o Grid Layout das CSS, utilizando sua tecnologia para uma responsividade de listas sem nenhuma Media Query.

A grade onde os produtos são colocados consegue calcular automaticamente o número de itens que cabem em uma linha, de tal forma que ela mude a organização de modo fluido e organizado.

Kami

9/01/2026

É o meu maior e mais ambicioso projeto, sendo uma aplicação mobile feita em Python com o objetivo de criar uma rede social de postagem de resumos escolares entre um grupo de 6 amigas. Usando as bibliotecas Kivy e Kivy MD, foi uma forma de entender como as mais diversas redes sociais funcionam

Como um projeto amador, estou ciente que possuí diversos erros e possíveis otimizações, mas só de ver ele pronto e devidamente funcionando compensou toda a jornada de 6 meses produzindo ele, na qual trouxe uma nova amizade, aprendizado e superação.
